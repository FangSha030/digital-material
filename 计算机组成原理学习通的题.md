# 分析题汇总

## 1. (分析题, 30分)
**题干**：定点数的运算，设数值位为7位，`x = 83`，`y = -99`，用变形补码求 `x+y` 与 `x-y`，并判断是否溢出。

**计算过程与结果**：
1.  **转换为二进制**：
    *   $X = 83 = 1010011_2$
    *   $Y = -99 = -1100011_2$
2.  **变形补码计算**：
    *   **求 $x+y$**：
        $[x]_{补}' = 00\ 1010011$，$[y]_{补}' = 11\ 0011101$
        相加：$00\ 1010011 + 11\ 0011101 = 11\ 1110000$（最高位丢掉，双符号位相同为 `11`，**无溢出**）
    *   **求 $x-y$（即 $x+(-y)$）**：
        $[x]_{补}' = 00\ 1010011$，$[-y]_{补}' = 00\ 1100011$
        相加：$00\ 1010011 + 00\ 1100011 = 01\ 0110110$（最高位丢掉，双符号位相异为 `01`，**溢出，正溢出**）

**知识点**：
数据表示与运算、数制与编码、定点数的表示与运算、进位计数制及相互转换、真值与机器数、定点小数、定点加减法运算、溢出判断（双符号位判断）

## 2. (分析题, 40分)
**题干**：假设机器字长为8位，将十进制数 `79` 与 `-79` 表示成原码、反码、补码，再将其分别算术左移1位、右移1位并还原真值，再分析其结果的正确性。

**计算过程与结果**：
1.  **机器数表示（8位，含1位符号位）**：
    | 十进制数 | 原码       | 反码       | 补码       |
    | :------- | :--------- | :--------- | :--------- |
    | 79       | `01001111` | `01001111` | `01001111` |
    | -79      | `11001111` | `10110000` | `10110001` |
2.  **移位操作与真值还原（以 -79 为例）**：
    | 移位操作 | 机器数（反码） | 对应真值 | 分析             |
    | :------- | :------------- | :------- | :--------------- |
    | 移位前   | `1110000`      | -79      | 基准值           |
    | 左移1位  | `11100001`     | -30      | 左移丢0，**结果错误** |
    | 右移1位  | `11011000`     | -39      | 右移丢0，**影响精度** |
    | 移位操作 | 机器数（补码） | 对应真值 | 分析             |
    | :------- | :------------- | :------- | :--------------- |
    | 移位前   | `1110001`      | -79      | 基准值           |
    | 左移1位  | `11100010`     | -30      | 左移丢0，**结果错误** |
    | 右移1位  | `11011000`     | -40      | 右移丢1，**影响精度** |

**结论**：负数反码/补码左移时数值位高位丢0会导致结果错误，右移时低位丢0/1会影响精度。

**知识点**：
数据表示与运算、数制与编码、定点数的表示与运算、真值与机器数（原码/反码/补码）、算术移位、精度分析
